# Sketch Pad

## Project Overview

Sketch Pad is a web-based application that allows users to create digital drawings directly in their browser. It is built using **JavaScript**, **HTML**, and **CSS**.

### Features
- Draw with various brush sizes and colors.
- Clear your sketchpad with a single click.
- Save your drawings.

### Technologies Used
- **JavaScript** for dynamic interactions and drawing functionalities.
- **HTML** for the layout and structure of the application.
- **CSS** for styling and presentation.

### Getting Started
To run the Sketch Pad application, simply clone the repository and open the `index.html` in a web browser.
